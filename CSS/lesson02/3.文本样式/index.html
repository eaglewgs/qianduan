为进一步完善英文课程体系，拓展学生国际化视野，提升专业学习能力，营造校园国际化氛围，学校国际交流合作处联合研究生院、 教务处， 推出系列国际课程。

学校拟邀请国外知名高校的外籍教师全英文授课，教学方式采用在线教学、课堂教学等多种方式， 每门课程2学分。 国际课程作为选修课，学分计入研究生培养计划总学分内
